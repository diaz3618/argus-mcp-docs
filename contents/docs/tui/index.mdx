description: "Interactive terminal UI for Argus MCP with real-time monitoring, capability browsing, and server control."
---

Argus includes an interactive terminal UI built with [Textual](https://textual.textualize.io/). The TUI connects to a running Argus server via the management API and to the argusd daemon via Unix Domain Socket, providing real-time monitoring, capability browsing, container/pod management, and server control.
