---
title: "Registry"
description: "Browse, search, and install MCP servers from remote catalogs, with automatic config sync for zero-downtime updates."
---

The Registry feature lets you browse, search, and install MCP servers from one or more remote catalogs directly inside the TUI or programmatically via the registry client. Combined with Config Sync, newly installed servers start connecting without a restart.

## In This Section

- [Overview](/docs/registry/overview) -- Registry architecture, API contract, configuration, installation flow, caching, and running your own registry.
- [Sync](/docs/registry/sync) -- Hot-reload and change detection for zero-downtime configuration updates.
