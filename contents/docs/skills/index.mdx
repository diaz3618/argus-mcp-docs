Skills are **portable, self-contained bundles** of MCP tools, workflows, and configuration. They act as a packaging mechanism: instead of manually adding backends one-by-one, you install a skill that declares everything a particular use-case needs.

<Note>
Skills are discovered automatically from the `skills/` directory in the project root, or from paths defined in your configuration.
</Note>
