Argus MCP uses a structured YAML configuration file to define backends, authentication, middleware, and operational settings.

This section covers all configuration topics:

- [Overview](/docs/configuration/overview) -- File format, config structure, variable expansion, and server settings.
- [Backends](/docs/configuration/backends) -- Backend types (stdio, SSE, streamable-http), timeouts, filters, and tool overrides.
- [Container Isolation](/docs/configuration/container-isolation) -- Automatic container isolation for stdio backends.
- [Authentication](/docs/configuration/authentication) -- Incoming client authentication modes.
- [Authorization](/docs/configuration/authorization) -- RBAC policy enforcement.
- [Secrets Management](/docs/configuration/secrets-management) -- Encrypted secret store and providers.
