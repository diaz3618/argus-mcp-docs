## Bug Fixes

### BUG-01 — Container-Isolation 404 Fix

**Root cause:** Not a nav href error — `contents/docs/configuration/container-isolation/index.mdx` compiled with errors due to MDX-unsafe patterns in GFM table cells (bold+backtick nesting, angle brackets in inline code). `getDocument()` returned null → `notFound()` → 404 page rendered.

**Fix:** Rewrote two table cell descriptions (lines 184, 185) to plain prose. No structural changes to routes, settings, or framework.

**Acceptance criteria — all met:**
- `/docs/configuration/container-isolation/` renders without 404 (212KB real content) ✓
- `out/docs/configuration/container-isolation/index.html` present in static export ✓
