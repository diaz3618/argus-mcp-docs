Go transport pattern (from RESEARCH.md):

Docker direct pass-through pattern (from github.yaml):
```yaml
# command: docker + args[0] = "run" → Argus treats as already-containerized
# Do NOT add container: key — no Argus wrapping needed
command: docker
args:
  - "run"
  - "-i"
  - "--rm"
  - "image-name"
```

source_url build constraints (from schema validator):
- source_url must use https:// or git+ssh:// scheme
- build_steps REQUIRED when source_url is set
- entrypoint REQUIRED when source_url is set
- build_steps entries CANNOT contain: backticks, $(), ()
- entrypoint elements CANNOT contain: ; & | ` $ ( ) { } [ ] < > ! # ~ \ newline

Transport auto-detection for source_url:
- If any of "npm ", "npx ", "node " appear in build_steps OR entrypoint starts with "node" → node:22-alpine base image
- Otherwise → python:3.13-slim base image
